# Smart Restaurant POS System

A terminal-based interactive Cashier and Point of Sale (POS) system built in Python.

##  Features

- **View Menu:** Display the restaurant's complete menu, including both food meals and beverages.
- **Add to Order:** Choose any item and add it to the current customer order by typing its specific ID.
- **View Order:** View the current items added to the order along with their base prices.
- **Print Receipt:** Calculate and display the final receipt, automatically applying specialized item taxes and discounts (e.g., Happy Hour discounts or service fees).
- **Graceful Exit:** Allow the cashier or user to exit the program smoothly through the menu interface.
